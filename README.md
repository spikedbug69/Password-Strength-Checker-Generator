 Password Strength Checker & Generator (GUI)

A Python-based GUI application that evaluates password strength using **entropy-based analysis**, estimates **realistic crack time**, and securely **generates strong passwords** using cryptographic randomness.

This project focuses on **security awareness fundamentals**, not superficial strength checks.
